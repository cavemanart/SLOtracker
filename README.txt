SRE Metrics Analyzer - Streamlit App

Setup Instructions:
1. Install dependencies:
   pip install streamlit pandas matplotlib fpdf

2. Run the app:
   streamlit run sre_metrics_app.py

Features:
- Define and save SRE metrics (SLA, SLO, SLI, KPI, KPR)
- View metrics in a dashboard
- Export metrics to CSV and PDF
